1. https://icomoon.io/ 접속
2. 오른쪽 상단 IcoMoon App 클릭
3. 오른쪽 상단 Untitled Project 1(프로젝트 명) 옆 아이콘 클릭
4. Import Project 클릭
5. selection.json 임포트
6. 하단 Untitled Project 1 Load 클릭
7. 왼쪽 상단 Import Icons 클릭, 원하는 svg 이미지 임포트 후 들어있는 아이콘 전부 선택한 후 오른쪽 하단에 Generate Font 클릭
8. 새로 넣은 아이콘 명칭 변경 혹은 그대로 한 다음(원래 있던 아이콘들은 절대 이름바꾸지 않기) 오른쪽 하단 Download 눌러 zip파일 다운 받기
9. zip파일에 있는 파일들(font,json) 원래 있던 파일에 덮어씌우기 혹은 바꿔넣기
